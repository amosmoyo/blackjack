Introduction

Blackjack is a simple webApp that resembles the normal Rummy Card Game.You are the player while the Computer is the Dealer.

How to play

Every time you start a new game round, the virtual cards get shuffled. Each card has an associated value.The Start or Start New Game Button Indicates the start of a new game or game round. TheHit Button allows you to pick a card.The Stay Button indicate that you are not picking a card.

How to win

The maximum score the dealer or the player can have is 21. If the dealers value is greater than 21 then the Player has won and vise versa. if(playerScore > dealerScore) and the difference in there point is less than 3 then the Player is said to have won and vice versa.
